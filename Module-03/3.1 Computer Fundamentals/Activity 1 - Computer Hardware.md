## Activity Objective

The objective for this exercise is for students to be able toÂ identify and describe the function of various hardware components of a computer system, including the CPU, motherboard, RAM, storage devices (HDD and SSD), power supply unit (PSU), graphics processing unit (GPU), input devices (keyboard, mouse, touchpad), output devices (monitor, speakers, printer), cooling system, and expansion cards. Students will also demonstrate an understanding of how these components work together to perform tasks and execute programs on a personal computer.



## Activity Instructions

Complete the knowledge check, and identify the purpose and function of each hardware component.
